# Lambda Players - L4D2 Weapon Pack
Adds over 27 new weapons from Left 4 Dead 2 to choose for [Lambda Players](https://github.com/IcyStarFrost/Lambda-Players/). Doesn't require Left 4 Dead 2 to be mounted.

### Weapon List
Currently this adds these weapons:
 - Pistol (P220)
 - Magnum (Desert Eagle)
 - SMG (UZI)
 - Silenced SMG (MAC-10)
 - Pump Shotgun
 - Chrome Shotgun
 - Auto Shotgun
 - Combat Shotgun (SPAS-12)
 - Assault Rifle (M16)
 - AK-47
 - Desert Rifle (SCAR)
 - Hunting Rifle
 - Military Rifle
 - M60
 - Grenade Launcher
 - Fireaxe
 - Crowbar
 - Frying Pan
 - Electric Guitar
 - Baseball Bat
 - Cricket Bat
 - Machete
 - Katana
 - Chainsaw
 - Pitchfork
 - Shovel
 - Tonfa
